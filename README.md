# GF2 Companions AIO

All-in-one companion mod for **S.T.A.L.K.E.R. Anomaly / GAMMA** bringing characters from **Girls Frontline 2: Exilium** into the Zone as recruitable companions.

---

## Characters

| Name | Default Faction | Spawn Zone |
|------|----------------|------------|
| Alva | Duty | Bar |
| Voymastina | Duty | Bar |
| Helen | Freedom | Army Warehouses |
| Leva | Bandits | Dark Valley |
| Vector | Military | Agroprom |
| Lind | Ecologist | Yantar |
| Klukai | Mercenary | Dead City |
| Nikketa | ISG | Darkscape |
| Phaetusa | SIN | Red Forest |
| Robella | Clear Sky | Great Swamps |
| Basti | Renegade | Meadow |
| Groza | Monolith | Pripyat |
| Mosin | Loners | Cordon |
| Cheeta | Ecologist | Yantar |

Each companion is **invulnerable** (UNLESS shot by the player), can be recruited and dismissed, and returns to their home smart terrain when dismissed.

---

## Features

- Everything from the standalone characters
- Clean FOMOD installer: pick individual characters and a faction preset independently

---

## Requirements

- S.T.A.L.K.E.R. Anomaly 1.5.2+
- GAMMA and Mod Organizer 2

---

## Installation
ModDB: Download the .rar, then add it through the MO2 UI "Install a mod from an archive" and follow the installation steps.
Github: Just download the zip, re-zip/rar the "GF2_AIO" folder, then add it through the MO2 UI "Install a mod from an archive" and follow the installation steps.

Start a new game after (re)installing, issues may occur when changing character's factions during a playthrough.
